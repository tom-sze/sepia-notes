About

Technology

Sepia Notes is a web app, built with the React javascript framework. To allow for seamless git merges, we convert the user-generated rich text to markdown under the hood.

Privacy Policy

We do not collect user data, or serve advertisements. The only 3rd party connection is to a CORS (cross-origin resource sharing) server, which enables us to perform git fetch, and push operations from within the browser.

Who we are

Although this app is a one-person project, I do my best to squash bugs, optimize performance, and introduce new features periodically.

Some major features are scheduled to arrive in the coming weeks, but I will always prioritize development in a way that ensures Sepia Notes remains free.
