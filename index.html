About

Technology

Sepia Notes is a web app, built with the React javascript framework. We use React version 19 (with compiling) to allow for optimal performace.

Most of the developement time thus-far has gone towards creating the rich-text editor. Now that this part of the project is completed, other features will start ariving soon.

Privacy Policy

We do not collect user data, or serve advertisements. The only 3rd party connection is to a CORS (cross-origin resource sharing) server, which enables us to perform git operations from within the browser.

Who we are

Although this app is a one-person project, I do my best to squash bugs, optimize performance, and introduce new features periodically.

Some major features are scheduled to arrive in the coming weeks, but I will always prioritize development in a way that ensures Sepia Notes remains free.
